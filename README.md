# 🍝 Pasta Studio® — Landing Page

Landing page de lanzamiento para **Pasta Studio**, un local de pastas en Jesús María, Córdoba.

Sitio de una sola pantalla con countdown regresivo, scroll snap entre secciones, navegación fija con glassmorphism y diseño 100% alineado al branding de la marca.

🔗 **Live**: [pastastudio.netlify.app](https://pastastudio.netlify.app/)

![Pasta Studio Preview](/public/images/og-image.png)

---

## Stack

| Tecnología | Detalle |
|---|---|
| **Vite 7** | Build tool |
| **Vanilla JS** | Sin framework |
| **CSS puro** | Custom properties, scroll snap, clamp() |
| **Netlify** | Deploy automático desde GitHub |
